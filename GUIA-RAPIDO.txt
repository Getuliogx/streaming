MINHA STREAM SIMPLES — GUIA RÁPIDO

NÃO TEM POSTGRESQL.
NÃO PRECISA EDITAR catalog.json.

1) Envie esta pasta para um repositório do GitHub.

2) Crie um token do GitHub com acesso somente a esse repositório e permissão para ler e gravar conteúdo.

3) No Render, crie um Web Service ligado ao repositório.

Build Command:
npm ci

Start Command:
npm start

4) No Render, adicione:

ADMIN_PASSWORD = sua senha do painel
GITHUB_TOKEN = seu token do GitHub
GITHUB_REPO = usuario/repositorio
NODE_ENV = production

5) Abra:
https://seu-site.onrender.com/admin

6) Digite a senha, cole o título, o link do vídeo e a capa. Clique em SALVAR.

O painel salva sozinho no GitHub em um ramo separado chamado "catalogo".
Você não precisa abrir nem editar arquivos.

LINKS ACEITOS:
- OK.ru
- Google Drive
- link direto MP4 do servidor do seu PC
- HLS/M3U8

IMPORTANTE:
O token fica somente nas variáveis do Render. Nunca coloque o token dentro dos arquivos do projeto.
